# EDA Project

Link to dataset:
[Shopping Behavior & Preferences Study](https://www.kaggle.com/datasets/ranaghulamnabi/shopping-behavior-and-preferences-study)

## Column Descriptions

* Column 1: description of column

Customer ID - unique number order/index
Age - age of customer
Gender - male or female
Item Purchased - what did they buy (ex. pants, dress, jewelry)
Category - what kind of thing they bought (ex. clothing, accessories)
Purchase Amount - how much the item cost
Location - location of the buyer
Size - size of item
Color - color of item
Season - what season the item was bought in
Review Rating - customer review out of 5
Subscription Status - true/false are they subscribed (ex. amazon prime)
Shipping Type - how they chose the item to be delivered
Discount Applied - true/false if they applied a discount
Promo Code Used - true/false if they used a promo code
Previous Purchases - number of previous purchases
Payment Method - method of payment for the item
Frequency of Purchases - how often they purchase items

## Hypotheses/Questions

1. Is there a correlation between seasons and clothing item purchased?
2. Is there a correlation between age and item color?
3. Is there a correlation between item color and season?
4. Is there a correlation between location and purchase amount?
5. Is there a correlation between gender and frequency of purchase?
